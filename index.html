Politique de Confidentialité

PENSÉES ACTUELLES
Dernière mise à jour : 17 Avril 2026

Introduction

Bienvenue dans l'application « PENSÉES ACTUELLES » Cette politique de confidentialité a pour but de vous informer de manière transparente sur la manière dont nous traitons vos données personnelles lorsque vous utilisez notre application.

Nous accordons une importance capitale à la protection de votre vie privée. Conformément au Règlement Général sur la Protection des Données (RGPD), nous nous engageons à respecter la confidentialité de vos informations.

Notre engagement : Aucune donnée personnelle n'est collectée, stockée ou partagée via nos serveurs. Toutes les informations restent sur votre appareil.

Collecte des informations

Nous ne collectons AUCUNE donnée personnelle vous concernant. L'application fonctionne entièrement en local sur votre appareil Android.

Cependant, certaines informations techniques peuvent être utilisées de manière strictement anonyme pour le bon fonctionnement de l'application :

  • Données de progression de lecture : stockées localement pour mémoriser votre dernière page lue. Ces données ne sont jamais transmises à des serveurs externes.
  • Préférences d'affichage : votre choix de thème (clair/sombre) est sauvegardé uniquement sur votre appareil.
  • Informations techniques anonymes : Aucune information comme l'adresse IP, modèle d'appareil ou identifiant publicitaire n'est collectée.
Aucun compte requis Aucune inscription Aucune donnée partagée

Utilisation des données

Les seules données stockées sont utilisées exclusivement pour :

  1. Mémoriser votre progression de lecture : afin de vous permettre de reprendre la lecture là où vous vous êtes arrêté.
  2. Sauvegarder vos préférences visuelles (thème clair/sombre) pour une expérience personnalisée.

Aucune exploitation commerciale, analyse comportementale ou recoupement avec des données externes n'est effectuée. Vous gardez le contrôle total sur ces informations.

Suppression des données : Vous pouvez effacer toutes les données stockées par l'application à tout moment via les paramètres de votre appareil (Paramètres → Applications → Pensées Actuelles → Stockage → Effacer les données).

Partage des données

Nous ne partageons aucune donnée avec des tiers. Étant donné que nous ne collectons aucune information personnelle, il n'y a rien à partager.

L'application n'intègre aucun SDK de suivi publicitaire, aucun outil d'analyse externe (Google Analytics, etc.), ni aucun réseau social. Toutes les ressources (images, textes) sont intégrées et chargées depuis des CDN sécurisés sans collecte de données.

Aucune publicité ciblée – L'application ne contient pas de publicités de tiers.

Sécurité des données

Bien que nous ne collections aucune donnée, nous mettons en œuvre des mesures de sécurité robustes pour protéger l'intégrité de l'application et empêcher toute intrusion :

  • Les données locales sont stockées dans l'environnement sécurisé du stockage local d'Android.
  • L'application ne demande aucune permission sensible (localisation, contacts, fichiers média, etc.).
  • Les communications externes sont limitées au chargement des images depuis un serveur sécurisé (HTTPS).

Vos droits

Conformément au RGPD, vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès : vous pouvez demander à connaître toutes les données que nous détenons sur vous. (Aucune donnée n'est détenue).
  • Droit de rectification : vous pouvez corriger toute information inexacte.
  • Droit à l'effacement : vous pouvez supprimer toutes les données locales via les paramètres de l'application ou de votre appareil.
  • Droit à la limitation du traitement : vous pouvez restreindre l'utilisation de vos données (sans objet ici).
  • Droit à la portabilité : vous pouvez récupérer vos données dans un format structuré.

Pour exercer ces droits, veuillez nous contacter à l'adresse mentionnée ci-dessous. Nous répondrons dans les meilleurs délais (dans un délai maximum d'un semaine).

Protection des enfants

Notre application s'adresse à un public général et ne collecte sciemment aucune information personnelle d'enfants de moins de 13 ans. Si vous êtes parent ou tuteur et pensez que votre enfant nous a fourni des données personnelles, veuillez nous contacter immédiatement. Aucune donnée n'étant collectée, ce risque est inexistant.

L'ensemble du contenu de l'application « Pensées Actuelles » est approprié pour tous les âges.

Transferts internationaux

Comme nous ne collectons aucune donnée personnelle, aucun transfert international de données n'a lieu. Les images utilisées dans l'application sont hébergées sur des services de stockage d'images situés dans l'Union Européenne, mais cela ne concerne que des fichiers publics non associés à des données personnelles.

Cookies

Cette application n'utilise aucun cookie de suivi. Les seules données stockées localement (localStorage) servent uniquement à des fins fonctionnelles (progression et thème). Aucun cookie tiers n'est déposé.

Vous pouvez à tout moment vider le stockage local de l'application via les paramètres de votre navigateur ou d'Android sans affecter le fonctionnement de base.

Modifications de cette politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité pour refléter les évolutions de l'application ou des exigences légales. Toute modification sera publiée dans l'application et, si elle est substantielle, nous vous en informerons via une notification dans l'application.

La date de la dernière mise à jour est indiquée en haut de cette page. Nous vous encourageons à consulter régulièrement cette politique.
